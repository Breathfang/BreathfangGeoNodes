Description
-----------

Bend the mesh around axis.

-----

.. toctree::
    :maxdepth: 2
    :caption: Contents:

.. image:: example.jpg
   :target: example.jpg
   :width: 400px

Input
-----

* **Geometry**: Mesh input.
* **Axis**: Axis to bend around.
* **Axis Direction**: Vector to bend around.
* **Center**: Center of the bend.
* **Angle**: Angle to bend.
* **Lock Center?**: Lock the center of the bend.

Output
------

* **Geometry**: Mesh output
