HOW TO INSTALL DRAGONGRAPH TOOLSET PACK
=====================================================

1. Locate the downloaded DragonGraph's Toolset Pack <version>.zip
2. Extract the DragonGraph's Toolset Pack <version>.blend in any location
3. Open File -> Preferences -> File Paths -> Asset Libraries
4. Click (+) Add Asset Library
5. Locate the DragonGraph's Toolset Pack folder and select DragonGraph's Toolset Pack <version>.blend.
6. Click Add Asset Library

Note: Make sure you've set Import Method as Append (Reuse Data) to prevent breaking changes when updating




HOW TO UPDATE DRAGONGRAPH GEOMETRY NODES TOOLSET PACK
====================================================

1. Here is the steps to update DragonGraph’s Toolset Pack:
2. Extract the DragonGraph's Toolset Pack <new version>.zip
3. Locate DragonGraph's Toolset Pack <new version>.blend in extracted folder, then press CTRL + C or right click -> Copy
4. Open your directory that contains DragonGraph's Toolset Pack <old version>.blend
5. Rename the DragonGraph's Toolset Pack <old version>.blend to DragonGraph's Toolset Pack <old version>.blend.old, then press CTRL + V or right click -> Paste.
6. (Optional) You can delete DragonGraph's Toolset Pack <old version>.blend.old.
7. Restart Blender

Warning: If you set Import Method as Link, you need to set Import Method as Append (Reuse Data) first. Otherwise it will break your project file.



HOW TO REINSTALL DRAGONGRAPH GEOMETRY NODES TOOLSET PACK
=======================================================
If you want to reinstall DragonGraph’s Toolset Pack, this is same as how to update DragonGraph’s Toolset Pack.

Just download the same version as you’ve downloaded before, and then extract it.



HOW TO UNINSTALL DRAGONGRAPH GEOMETRY NODES TOOLSET PACK
=======================================================

1. Open File -> Preferences -> File Paths -> Asset Libraries
2. Select DragonGraph's Toolset Pack <version>.blend and click Remove

Note: You can delete your package folder to free up space of your disk.
